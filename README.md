깃클론 (이부분 모르시는 분들은 말씀해주세요!)
클론한 레포지토리 들어가서 터미널에 npm install -f
터미널에 npx playwright install 입력
설치가 다 완료되면 npm run dev 입력
터미널에 다음과 같은 메세지가 뜨면 로컬서버 open
